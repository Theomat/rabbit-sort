Game Rules

The goal of this game is to sort both the hat and rabbit tokens from left to right in ascending order. Both hats and rabbits tokens are numbered from 1 to 9.
You will only see 9 stacks, hat tokens are stacked over rabbit tokens, so you cannot see the number on the rabbit tokens.

You have 2 minutes and 3 actions in order to sort them:

    • "Peeking" shows you the rabbit token underneath the hat token.
      "Swapping hats" swaps the hats of two stacks.
      "Swapping all" swaps the hats and rabbits tokens of two stacks.
    • You can use the number keys to directly select the stack with the hat token whose number match the one pressed.

    Difficulty

    Difficulty up to level 4 add one lock per level, a locked stack cannot be selected for any action.
    Difficulty level 5 add a random swap of stacks every 3 moves but removes all locks.
    Difficulty after level 5 add one lock per level up to 3.
    And the last two difficulty level each remove 30 seconds.

    Note on random swaps

    Since random swaps with locks can be very hard to sort because of bad luck, once the hat tokens and rabbit tokens are correctly paired, swaps no longer occur.
